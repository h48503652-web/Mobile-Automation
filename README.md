# Mobile Automation Project - Appium & Java

This project demonstrates mobile automation testing on Android devices using **Appium**, **Java**, and **JUnit 5**. It covers two main tasks: automating the Clock application and interacting with the Android Settings menu.

## 🚀 Features
* **Page Object Model (POM):** Clean and maintainable code structure.
* **Generic BaseTest:** A flexible infrastructure that supports multiple apps.
* **Smart Scrolling:** Advanced scrolling mechanism to find elements in long lists.
* **Task 1 - Clock App:** Automates setting a timer for 10 seconds and validating the input.
* **Task 2 - Settings App:** Automates navigation and interaction within the Android Settings.

## 🛠 Tech Stack
* **Language:** Java
* **Automation Tool:** Appium (v2.x)
* **Driver:** UiAutomator2
* **Test Runner:** JUnit 5
* **Build Tool:** Maven



## 📋 Prerequisites
1. **Node.js** installed.
2. **Appium Server** installed: `npm install -g appium`.
3. **Android Driver:** `appium driver install uiautomator2`.
4. **Android Studio** with an Emulator (Recommended: Android 9.0 or higher).
